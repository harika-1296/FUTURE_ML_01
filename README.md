# Sales & Demand Forecasting using Machine Learning

## 📌 Project Overview
This project predicts future sales using historical data. It helps businesses make better decisions related to inventory, staffing, and planning.

## 🎯 Objective
To forecast future sales based on past trends using Machine Learning techniques.

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Streamlit (for dashboard)

## 📊 Key Steps
- Data cleaning and preprocessing
- Time-based feature engineering
- Sales trend visualization
- Model building using Linear Regression
- Future sales prediction

## 📈 Results
- Sales show fluctuations with an overall increasing trend
- Forecast predicts steady growth in future sales

## 📊 Dashboard

An interactive dashboard was built using Streamlit to visualize sales trends and insights.

### Features:
- 📈 Monthly sales trend visualization  
- 💰 KPI metrics (Total, Average, Max Sales)  
- 📅 Date filtering for analysis  
- 📊 Data preview for better understanding

## ▶️ How to Run the Project

1. Install required libraries:
pip install pandas numpy matplotlib scikit-learn streamlit

2. Run the dashboard:
streamlit run app.py

- Helps in inventory planning

- Improves decision-making
